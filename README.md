# Projeto TikT0k (React Native)

Projeto criado para aprendizado sobre react native, utilizando expo.

## Bibliotecas utilizadas:
### - React Navigation
### - Expo-av

## Instalação

Clone esse repositório e instale as dependencias do projeto:

```bash
  npm install
```
    
Depois rode o projeto:
```bash
  npx expo start
```
